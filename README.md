# Практична робота №13
## Серіалізація об’єктів в Java

У даній практичній роботі реалізовано серіалізацію об’єктів у Java на основі практичної роботи №11.

## Реалізовано:
- серіалізацію класів `UserIdentifier` та `User`;
- збереження колекції користувачів `HashMap<UserIdentifier, User>` у файл;
- відновлення користувачів з файлу;
- автоматичне оновлення `nextId` після відновлення;
- скидання стану логування після десеріалізації;
- консольне меню для демонстрації роботи програми;
- запит на відновлення бази користувачів під час запуску програми;
- запит на збереження користувачів перед завершенням програми.

## Особливості реалізації
- `UserIdentifier` та `User` реалізують інтерфейс `Serializable`;
- поле `isLoggedIn` у класі `User` позначено як `transient`, тому воно не серіалізується;
- збереження та відновлення виконуються через `ObjectOutputStream` та `ObjectInputStream`.

## Використані технології
- Java
- Serializable
- ObjectOutputStream
- ObjectInputStream
- HashMap
- LinkedList
- Comparator
- Predicate
